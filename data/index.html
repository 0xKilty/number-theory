About

number-theory is an open source library designed to provide educational programming examples of functions in number theory. This project is avalible in python though pip modules and can be viewed on PyPi.

Installation and Usage

To install numbertheory via pip: 

        pip install numbertheory
To import numbertheory in a python script: 
        import numbertheory as nt
To use numbertheory in a python script: 
        nt.gcd(31415, 926535)
