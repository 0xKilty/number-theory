Modular exponentiation

int mod_exp(int base, int power, int mod) #

Uses the fast powering algorithm to exponentiate a number modulus.

\( a^b \equiv x \mod m \)

        
            int mod_exp(int base, int power, int mod) {
                int temp = base;
                int res = 1;
                for (int i = 0; i < 32 - __builtin_clz(power); i++) {
                    if ((power >> i) & 1)
                        res = (res * temp) % mod;
                    temp = (temp * temp) % mod;
                }
                return res;
            }
